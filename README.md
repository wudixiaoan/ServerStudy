
=================== 开发环境搭建 ========================
JDK 安装配置:  
增加环境变量:JAVA_HOME  
C:\Program Files (x86)\Java\jdk1.7.0_02\
----------------------------------------------------
Tomcat 安装配置:  
增加环境变量:CATALINA_HOME  
F:\apache-tomcat-8.5.34  
Path中增加:  
%CATALINA_HOME%\bin  
----------------------------------------------------
Maven 安装配置:  
增加环境变量:MAVEN_HOME  
F:\apache-maven-3.6.0  
Path中增加:  
%MAVEN_HOME%\bin  
----------------------------------------------------
